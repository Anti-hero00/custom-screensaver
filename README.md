custom-screensaver
==================

Probably the only custom screensaver you'll ever need on webOS TV.

Requires root.

Installation
------------
This should be downloadadble in Homebrew Channel. Otherwise, there's an `ipk` in
GitHub releases to the right. You are on your own here.
